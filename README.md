## 💻 Cara Menggunakan

1. Clone repositori ini:

```bash
git clone https://github.com/Libra2694/AisyahBot-py.git
cd AisyahBot-py
pip install -r requirements.txt
```

2. Run

```bash
python AisyahBot.py
```
